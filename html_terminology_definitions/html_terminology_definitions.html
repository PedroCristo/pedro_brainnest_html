HTML Terminology & Definitions

  1. What do HTML and CSS stand for?

    Answer: HTML stands for HyperText Markup Language, and CSS stands for Cascading Style Sheets.

  2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?

    Answer: To add paragraphs of text to a webpage, you would use HTML.

  3. Would you rather use HTML or CSS, to change the font and background color of a button?

    Answer: To change the font and background color of a button, you would use CSS.

  4. What is an HTML tag?

    Answer: An HTML tag is a code that is used to mark the beginning and end of an HTML element.

  5. What are the three parts of an HTML element?

    Answer: The three parts of an HTML element are the opening tag, the content, and the closing tag. For example: <p>This is the content</p> The opening tag is <p>, the content is This is the content, and the closing tag is </p>.

  6. What is the purpose of the doctype declaration?

    Answer: The purpose of the doctype declaration is to specify which version of HTML is being used in the document. This allows the browser to interpret and render the HTML correctly.

  7. What is the HTML element?

    Answer: The HTML element is the root element of an HTML document. It encapsulates all of the content on the page, and serves as a container for the head and body elements.

  8. What is the purpose of the head element?

    Answer: The head element is used to contain information about the document that is not directly related to the content of the page. This includes things like the title of the document, links to external stylesheets and scripts, and meta information such as keywords and descriptions. The head element is represented by the <head> tag and should be the first element within the HTML element.

  9. What is the purpose of the body element?

    Answer: The body element is used to contain the content of the webpage that is visible to the user. This includes text, images, and other HTML elements that make up the page. The body element is represented by the <body> tag and should be the second element within the HTML element, after the head element.

  10. How do you create a paragraph in HTML?

    Answer: To create a paragraph in HTML, you can use the <p> element. For example: 

     <p>This is a paragraph.</p>

  11. How do you create a heading in HTML?

    Answer: To create a heading in HTML, you can use one of the heading elements: <h1>, <h2>, <h3>, <h4>, <h5>, or <h6>. The heading elements range in size, with <h1> being the largest and <

    being the smallest. For example: 

     <h1>This is a heading</h1>

  12. How many different levels of headings are there and what is the difference between them?

    Answer: There are six levels of headings in HTML, numbered from 1 to 6. The difference between the heading levels is the relative size of the text. Heading level 1 <h1> is the largest, and heading level 6 <h6> is the smallest.

  13. What element should you use to make text bold and important?

    Answer: To make text bold and important, you can use the <strong> element. You can also use the <b> element for bold text, although the <strong> element is considered more semantically correct as it indicates that the text is important. For example: 

     <strong>This text is important and bold.</strong>

  14. What element should you use to make text italicized to add emphasis to it?

    Answer: To make text italicized to add emphasis to it, you can use the <em> element. You can also use the <i> element for italicized text, although the <em> element is considered more semantically correct as it indicates that the text should be emphasized. For example: 

     <em>This text is emphasized and italicized.</em>

  15. What relationship does an element have with any nested elements within it?

    Answer: An element that has other elements nested within it is called the parent element, and the nested elements are called child elements. The parent element and child elements have a parent-child relationship.

  16. What relationship do two elements have if they are at the same level of nesting?

    Answe: Two elements that are at the same level of nesting are called sibling elements. Sibling elements have a sibling relationship.

  17. How do you create HTML comments?

    Answer: To create an HTML comment, you can use the <!-- and --> tags. Anything you write between these tags will be treated as a comment and will not be displayed in the final webpage. For example: 

     <!-- This is an HTML comment -->

  18. What HTML tag is used to create an unordered list?

    Answer: To create an unordered list in HTML, you can use the <ul>` element. This element represents an unordered list of items, which are typically displayedwith bullet points. For example: 

                             
                          <ul>
                            <li> Item 1 </li>
                            <li> Item 2 </li>
                            <li> Item 3 </li>
                          </ul>

  19. What HTML tag is used to create an ordered list?

    Answer: To create an ordered list in HTML, you can use the <ol> element. This element represents an ordered list of items, which are typically displayed with numbers or letters. For example: 

                            
                          <ol>
                            <li> 1 . Item </li>
                            <li> 2 . Item </li>
                            <li> 3 . Item </li>
                          </ol>

  20. What HTML tag is used to create list items within both unordered and ordered lists?

    Answer: To create list items within both unordered and ordered lists in HTML, you can use the <li> element. This element represents a list item. For example: 

                            
                          <ul>
                            <li> Item 1 </li>
                            <li> Item 2 </li>
                            <li> Item 3 </li>
                          </ul>
            
                          <ol>
                            <li> 1 . Item </li>
                            <li> 2 . Item </li>
                            <li> 3 . Item </li>
                          </ol>

  21. What element is used to create a link?

    Answer : To create a link in HTML, you can use the <a> element. This element represents a hyperlink, which allows users to navigate to another webpage or a specific location on the same webpage. For example: 

     <a href="http://www.example.com">This is a link</a>

  22. What is an attribute?

    Answer: An attribute is a property of an HTML element that provides additional information about the element. Attributes are typically specified in the start tag of an element and take the form of a name-value pair. For example: 

     <a href="http://www.example.com">This is a link</a>

  23. What attribute tells links where to go ?

    Answer: The "href" attribute tells links where to go.

  24. What is the difference between an absolute and relative link?

    Answer: An absolute link specifies the full URL of a webpage or resource, while arelative link specifies the location of a webpage or resource relative to the current webpage.

  25. Which element is used to display an image?

    Answer: The "img" element is used to display an image.

  26. What two attributes do images always need to have?

    Answer: Images always need to have the "src" attribute (which specifies the source file for the image) and the "alt" attribute (which specifies alternative text for the image in case it cannot be displayed).

  27. How do you access a parent directory in a file path?

    Answer: You can access a parent directory in a file path by using "../" (without the quotes).

  28. What are the four main image formats that you can use for images on the web?

    Answer: The four main image formats that you can use for images on the web are JPEG, PNG, GIF, and SVG.
